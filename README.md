#DataPrepAI

DataPrepAI is a Python-based data preprocessing and quality analysis tool designed to automate one of the most time-consuming stages of the Machine Learning pipeline—data preparation.

The project helps users load datasets, analyze their quality, identify common data issues, and apply appropriate preprocessing techniques before model development. It aims to provide a structured and user-friendly workflow that simplifies exploratory data analysis and improves data reliability.

Features
Module 1: Dataset Loading & Overview
Load datasets from CSV and Excel files
Validate file paths and supported file formats
Display dataset overview including:
Number of rows and columns
Memory usage
Numerical and categorical column counts
Preview the dataset using the first five records
Module 2: Data Quality Analysis (In Progress)
Missing value analysis
Duplicate record detection
Data type validation
Constant column detection
High-cardinality analysis
Outlier detection
Correlation analysis
Data quality scoring
Smart recommendations for improving data quality
Module 3: Data Preprocessing (Planned)
Missing value handling
Duplicate removal
Outlier treatment
Feature encoding
Feature scaling
Data type conversion
Export cleaned dataset
